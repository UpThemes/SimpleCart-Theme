Getting Started with SimpleCart

SimpleCart is a simply magnificent shopping cart theme for WordPress, built mainly for use with WP E-commerce (but can be used for other purposes).

Installing SimpleCart

  1. Install the SimpleCart theme into the /wp-content/themes/ directory.
  2. Install the latest version of the WP E-commerce plugin.

Installing SimpleCart's Grid Layout for WP E-commerce

  1. To enable the grid-based product layout, you must install the included SimpleCart WP E-commerce theme into your /wp-content/plugins/wp-e-commerce/themes/simpleCart/ folder.
  2. Go to WP E-commerce Settings > Presentation and set "Theme" to "SimpleCart Theme"
  3. In WordPress, Go to Settings > Reading and change "Front Page Displays" to "A Static Page" and select the "Products Page" as the front page.
  4. Once you've done that, visit the SimpleCart Options page to set a custom logo.

Setting Up Products on your Homepage

  1. Go to Settings > Reading and change "Front Page Displays" to "A static page" and select your WP E-commerce product page.
